Revisão Geral

Lista de Tags aprendidas até aqui.

Title
Coloca um título na aba da página.
H1, H2, H3... H6
Títulos do Texto, os quais vão do nível um ao seis.
hr
Coloca linha horizontal na página.
p
Escreve parágrafos com texto.
br
Quebra a linha em um parágrafo.
link:favicon
Insere um favicon, um ícone, na aba da página.
img
Insere uma imagem.
strong
Coloca uma palavra em negrito.
em
Coloca uma palavra em itálico.
del
Risca uma palavra.
ins
Sublinha uma palavra.
code
Escreve um código de uma linguagem de programação.
pre
Escreve um texto de forma mono-espaçada; pré-formatação. É usada para organizar os códigos escritos com o comando escrito a cima
abbr
Escreve abreviações e seu significado.
big
Tag obsoleta usada para aumentar o tamanho do texto.
small
Usa-se para diminuir o tamanho do texto.
q
Escreve uma citação simples
blockquote
Escreve uma citação em bloco.
b
Tag obsoleta que coloca texto em negrito.
i
Tag obsoleta que coloca o texto em itálico.
mark
Coloca marcação em um texto.
sup
Coloca caractere ou palavra sobrescrito
sub
Coloca caractere ou palavra subscrito.
u
Tag obsoleta que escreve texto sublinhado.
bdo
Escreve texto invertido.
li
Adiciona um item/elemento de uma lista.
ol
Criar lista ordenada.
ul
Criar lista desordenada, ou seja, lista sem numeração, em que a ordem não importa.
dl
Cria uma lista de definições.
dt
Coloca o termo a ser definido em uma lista de definições.
dd
Adiciona definição de um item na lista de definições.
a
Coloca links.
